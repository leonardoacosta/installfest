② tmux-claude-usage

Coverage: FULL — home/dot_local/bin/executable_tmux-nexus-creds

Already rendered in status-right of every theme via #(tmux-nexus-creds), and the existing impl is superior for this setup:

  • Multi-account — nexus-agent tracks active_account + per-account 5H/7D (matches the BBAdmin/O365/Civalent/personal identity split). claude-usage is single-account.
  • Central agent vs claude-usage's shared-cache-clobber — every session writes one ~/.cache/claude-usage/usage file (its own code guards against the stale-replay flicker that causes).
  • Same cyan/yellow/red threshold UX already present.

Adopting it would be a regression.
